Cook-It or Book-It!

A simplifed way to solve the age old problem of deciding to dine out or stay in and cook!

With a single search command line, this app provides the user options between dining out or preparing a home cooked meal at the click of a button. This was achieved by:

- Creating a user friendly website utilizing Materialize design language
- Selecting two APIs that fit the application's needs (Zomato and Edamam Recipie Search)
- Building seperate AJAX functions for each API that return the desired results based off a single cuisine type
- Appending the results from each API on one page, allowing the user to easily compare options

  • Technologies Used:
  • HTML, CSS, jQuery, Materialize, AJAX
  • API's Utilized:
  • Zomato, Edamam Recipie Search
